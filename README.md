# Sistema de Gerenciamento de Quadras Esportivas

## Estrutura do Projeto
- `backend/` - API Node.js com Express e SQLite
- `frontend/` - Aplicação React com Electron

## Como executar
1. Clone o repositório
2. Execute `./setup.sh` para configurar dependências
3. Backend: `cd backend && npm run dev`
4. Frontend: `cd frontend && npm start`
5. Electron: `cd frontend && npm run electron`

## Desenvolvimento
- Fase 1: Core funcionalidades (4 semanas)
- Fase 2: Melhorias e otimizações (2 semanas)
